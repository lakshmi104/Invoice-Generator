# 🧾 Invoice Generator System

A simple web-based Invoice Generator built with **Python**, **Flask**, and **ReportLab**. Users can enter customer details, add multiple products, calculate GST automatically, and generate a professional PDF invoice.

## 🚀 Features

- Create invoices
- Add multiple products
- Automatic subtotal calculation
- Automatic GST (Tax) calculation
- Generate downloadable PDF invoices
- Simple and user-friendly interface
- Responsive design

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- ReportLab

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Invoice-Generator-Python.git
```

### 2. Navigate to the project folder

```bash
cd Invoice-Generator-Python
```

### 3. Install the required packages

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python invoice.py
```

### 5. Open your browser

```
http://127.0.0.1:5000
```
